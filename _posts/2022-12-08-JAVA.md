<h3>연산자</h3>
![](/assets/img/java/operator.png)

```java
연산자 중에  
 "="와 같이 쓰이는 연산자는 "="가 우측에 붙는다.

(자료형)은 형변환을 의미한다
int a = 10; 
(double)a -> 10.0

"!"연산자
부정의 의미.
A =ture; 라면 !A -> false

산술연산자
"%" : 모듈러스라고 부르며, 나머지를 구하는 연산자이다.

일반 논리 연산자
A && B && C               (vs)         A || B || C
모두 true => true                      하나라도 true => true
하나라도 false => false                모두 false => false

삼항 연산자
(조건식)? 식1 : 식2;
조건식이 참일 경우 식1,거짓일 경우 식2가 값으로 반환.
```
