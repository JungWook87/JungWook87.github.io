```html
<ul>
순서없는 목록을 작성하는 태그(영역)
```

```html
<li>(List Item)
목록에 들어가는 내용을 작성하는 태그
```

```html
<ol>(Ordered List)
순서있는 목록을 작성하는 태그

* ol 태그의 type 속성 *
기본값 : 1부터 1씩 증가하는 숫자
type="a" : 영어 소문자 순서
type="A" : 영어 대문자 순서
type="i" : 로마 소문자 순서
type="I" : 로마 대문자 순서

+ 추가속성
start="숫자" : 순서가 입력 숫자부터 시작
reversed : 순서가 반대로 카운트
```
