### 2. 요소 정렬 스타일(float / clear)

```html
✔️ float(띄우다) : 요소를 띄워서 좌/우로 정렬하는 속성
✔️ clear : float로 인해서 띄워져 있는 상태를 해제하는 속성
(float 사용 시 겹침 문제가 발생하는데 이를 해결할 수 있음)
```
